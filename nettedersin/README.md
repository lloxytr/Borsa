# Nettedersin (Sıfırdan Yeni Yapı)

Bu klasör, nettedersin.com için sıfırdan hazırlanmış yeni bir ön yüz başlangıç paketidir.

## İçerik
- `index.html`: Premium görünümlü ana sayfa + paketler
- `student.html`: Öğrenci paneli modül özeti
- `teacher.html`: Öğretmen paneli modül özeti
- `admin.html`: Admin paneli modül özeti
- `blog.html`: SEO amaçlı blog örneği
- `faq.html`: SSS sayfası
- `robots.txt` + `sitemap.xml`: Teknik SEO başlangıcı
- `assets/styles.css`: Tek font ailesi + 2 ana renk + 1 vurgu rengi tasarım sistemi

## Hızlı çalıştırma
```bash
cd nettedersin
python3 -m http.server 8080
```

Ardından: `http://localhost:8080`
